# Tic Tac Toe (C++)
## Overview
This is a simple implementation of the classic Tic Tac Toe game written in C++. The game allows two players to take turns marking the spaces in a 3Ã—3 grid, and the player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.
## Features
- Two-player mode
- Simple text-based user interface
- Checks for win, draw, or continue conditions
## Prerequisites
- A C++ compiler (e.g., g++)
